# Asthma Tagebuch Datenschutzerklärung

Statische Website für die Datenschutzerklärung der App `Asthma Tagebuch`.

## Dateien

- `index.html`: die eigentliche Website
- `styles.css`: Styling der Website

## Veröffentlichung

Die Seite kann direkt bei GitHub Pages, Netlify, Cloudflare Pages oder auf jedem normalen Webspace hochgeladen werden.

Wichtig vor dem Veröffentlichen:

1. Platzhalter für Name, Anschrift und E-Mail in `index.html` ersetzen.
2. Danach den Inhalt des Ordners unverändert hochladen.
3. Die öffentliche URL in der Google Play Console als Datenschutzerklärungs-URL hinterlegen.
