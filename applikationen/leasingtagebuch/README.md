# Leasing Tagebuch Datenschutzerklärung

Statische Website für die Datenschutzerklärung der App `Leasing Tagebuch`.

## Dateien

- `index.html`: die eigentliche Website
- `styles.css`: Styling der Website

## Veröffentlichung

Die Seite kann direkt bei GitHub Pages, Netlify, Cloudflare Pages oder auf jedem normalen Webspace hochgeladen werden.

Wichtig vor dem Veröffentlichen:

1. Den Inhalt des Ordners unverändert hochladen.
2. Die öffentliche URL in der Google Play Console als Datenschutzerklärungs-URL hinterlegen.
